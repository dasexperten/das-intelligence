# Contributing

Thank you for your interest in contributing to **DAS Intelligence Skills**.

This repository is intended to become a high-quality professional AI skills library. Contributions should be practical, structured, and useful for real business execution.

## Contribution Principles

Every contribution should be:

- specific, not generic;
- operational, not theoretical;
- structured, not random;
- testable, not vague;
- useful for commerce, marketing, marketplaces, SEO, sales, analytics, logistics, legal workflows, or business operations.

## Skill Quality Standard

Each skill should include:

- clear purpose;
- activation conditions;
- non-use conditions;
- expert role;
- operating procedure;
- checklists;
- templates;
- examples;
- evaluation tests;
- metadata.

## Pull Request Guidelines

Before opening a pull request:

1. Follow the repository folder structure.
2. Use clear Markdown.
3. Avoid placeholder text.
4. Include examples where possible.
5. Keep claims realistic and business-safe.
6. Do not add copyrighted material without permission.

## Naming Convention

Use lowercase kebab-case folder names:

```text
product-page-optimizer
wildberries-seo-specialist
tiktok-shop-creator-brief-builder
```
