# Director Daily Brief

## Purpose

Create a daily management brief with priorities, meetings, risks, open decisions, key numbers, and next actions.

## When to Use

Use this skill when preparing a daily director summary, morning brief, or end-of-day review.

## Inputs

Useful inputs: calendar, task list, messages, KPI notes, sales notes, operations updates, risks, deadlines, and open decisions.

## Workflow

1. Summarize current day context.
2. List top priorities.
3. Highlight risks, deadlines, and open decisions.
4. Summarize key numbers.
5. Recommend next actions.

## Output

Always provide daily summary, priority list, risks, decisions, KPI notes, and action list.

## Common Mistakes

- Too much detail.
- No priorities.
- No decisions.
- No action list.
