# Agent Router

## Purpose

Choose the right skill, agent, workflow, or tool path for a user request.

## When to Use

Use this skill when a task could belong to several categories such as marketing, website, SEO, marketplace, sales, operations, or analytics.

## Inputs

Useful inputs: user request, business area, urgency, files, desired output, tools, and constraints.

## Workflow

1. Identify the task type.
2. Match request to the best category and skill.
3. Detect missing inputs.
4. Choose workflow order.
5. Define expected output.

## Output

Always provide chosen route, skill list, input needs, workflow order, and output format.

## Common Mistakes

- Using one skill for everything.
- Ignoring missing inputs.
- Wrong category choice.
- No output format.
