# Market Entry Planner

## Purpose

Map files and data needed before launching products in a new country or sales channel.

## When to Use

Use this skill when preparing country entry, distributor onboarding, marketplace launch, or retail launch.

## Inputs

Useful inputs: country, channel, product category, product data, files, label needs, partner role, timeline, and local notes.

## Workflow

1. Define market and channel.
2. List category files.
3. Check label and product data needs.
4. Identify missing items and owner roles.
5. Create readiness plan.

## Output

Always provide file summary, file needs, label needs, missing items, owner notes, timing, and next actions.

## Common Mistakes

- No file map.
- No partner role.
- Missing label needs.
- No timing.
