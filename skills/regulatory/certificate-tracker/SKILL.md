# Certificate Tracker

## Purpose

Track certificates, declarations, test reports, manufacturer documents, and expiry dates for products and markets.

## When to Use

Use this skill when organizing product documents for registrations, distributors, retail buyers, marketplaces, or shipments.

## Inputs

Useful inputs: product, country, document type, issuing body, issue date, expiry date, owner, file link, and renewal notes.

## Workflow

1. List all documents by product and market.
2. Capture owner, file link, issue date, and expiry date.
3. Identify missing or expired documents.
4. Define renewal and follow-up actions.
5. Create status table.

## Output

Always provide document tracker, missing items, expiry notes, owner notes, and next actions.

## Common Mistakes

- No expiry tracking.
- Missing file links.
- No owner.
- Mixing documents by market.
