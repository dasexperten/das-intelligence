# Regulatory Skills

This folder contains practical product-rule and market-readiness skills for product files, labels, registrations, certificates, market entry, and update tracking.

These skills organize business information and do not replace local specialist review.

## Completed Regulatory Skills

1. `product-rules-planner`
2. `label-review-planner`
3. `registration-file-planner`
4. `certificate-tracker`
5. `market-entry-planner`
6. `update-tracker`

## Usage Logic

- product category rules → `product-rules-planner`
- label artwork or text → `label-review-planner`
- product file preparation → `registration-file-planner`
- certificates and dates → `certificate-tracker`
- market readiness → `market-entry-planner`
- changing requirements → `update-tracker`
