# Update Tracker

## Purpose

Track updates in product rules, documents, labels, market needs, and partner requests so the team can react on time.

## When to Use

Use this skill when a market, partner, marketplace, or retailer changes file or document expectations.

## Inputs

Useful inputs: market, source, date, update summary, affected products, required action, owner, deadline, and status.

## Workflow

1. Capture source, date, and update summary.
2. Identify affected products and channels.
3. List required actions and owners.
4. Set timing and status.
5. Add review rhythm.

## Output

Always provide update summary, affected products, action list, owner notes, timing, status, and next actions.

## Common Mistakes

- No source date.
- No owner.
- No affected product list.
- No review rhythm.
