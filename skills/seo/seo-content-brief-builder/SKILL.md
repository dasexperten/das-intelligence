# SEO Content Brief Builder

## Purpose

Create SEO content briefs that help writers produce pages matching search intent, buyer needs, expert standards, internal link strategy, and conversion goals.

## When to Use

Use this skill when the task is to prepare an SEO article brief, category page brief, product guide, comparison page, FAQ page, or landing page brief.

## Inputs

Useful inputs: target query group, search intent, audience, page goal, competitors, product facts, proof points, headings, FAQs, internal link targets, and CTA.

## Workflow

1. Define search intent and page goal.
2. Review top competitor content and gaps.
3. Build recommended title, meta, H1, heading outline, sections, FAQs, proof needs, internal links, and CTA.
4. Add quality requirements and things to avoid.
5. Provide final brief for writing.

## Output

Always provide content brief, outline, key questions, proof needs, internal links, metadata notes, CTA, and quality checklist.

## Common Mistakes

- Briefing writers with only keywords.
- No search intent.
- No proof requirements.
- No internal link plan.
- No conversion goal.
