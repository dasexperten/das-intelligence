# On-Page SEO Optimizer

## Purpose

Improve individual pages so they match search intent, communicate value clearly, support ranking, and help visitors take the next action.

## When to Use

Use this skill when the task is to optimize a specific page, category page, product page, landing page, article, or homepage for SEO.

## Inputs

Useful inputs: page URL or copy, target query group, search intent, page goal, title, meta description, headings, content, internal links, competitors, and conversion goal.

## Workflow

1. Define target query group and search intent.
2. Review title, meta description, H1, headings, content depth, internal links, and CTA.
3. Check if the page answers the query better than competitors.
4. Identify missing sections, proof, FAQs, and semantic terms.
5. Recommend on-page fixes and priority order.

## Output

Always provide on-page diagnosis, title/meta recommendations, heading structure, content improvements, internal link notes, FAQ ideas, and next actions.

## Common Mistakes

- Optimizing for keywords but not intent.
- Repeating terms unnaturally.
- Weak headings.
- No internal links.
- No conversion path.
