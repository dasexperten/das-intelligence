# Ecommerce SEO Expert

## Purpose

Improve organic search performance for ecommerce category, product, collection, brand, comparison, and informational pages.

## When to Use

Use this skill when the task is to optimize ecommerce SEO, category pages, product pages, product filters, product variants, merchandising pages, or commerce content.

## Inputs

Useful inputs: store URL, platform, product categories, page templates, keywords, Search Console data, product attributes, filters, variants, competitors, and conversion goal.

## Workflow

1. Define priority categories and commercial intent.
2. Review category, product, filter, variant, and collection page structure.
3. Check metadata, headings, copy, internal links, schema, indexation, and duplicate risks.
4. Map informational content to commercial pages.
5. Prioritize fixes by revenue potential and SEO impact.

## Output

Always provide ecommerce SEO diagnosis, category fixes, product page fixes, filter/indexation notes, schema notes, internal links, and priority roadmap.

## Common Mistakes

- Indexing weak filter pages.
- Product pages with thin content.
- Duplicate metadata across products.
- No category copy or internal links.
- Ignoring conversion while optimizing SEO.
