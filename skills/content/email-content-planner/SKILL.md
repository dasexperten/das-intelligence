# Email Content Planner

## Purpose

Plan email content that supports launches, retention, education, offers, customer updates, and B2B follow-up.

## When to Use

Use this skill when the task is to plan email campaigns, newsletters, sequences, product education emails, or follow-up messages.

## Inputs

Useful inputs: audience, goal, list segment, product, offer, proof points, objections, timing, tone, and CTA.

## Workflow

1. Define audience, goal, and segment.
2. Choose email type and message angle.
3. Build subject, preview, body structure, proof, and CTA.
4. Create sequence order when needed.
5. Add KPI and testing notes.

## Output

Always provide email plan, subject ideas, message outline, CTA, sequence notes, KPI notes, and next steps.

## Common Mistakes

- Sending without segment logic.
- Weak subject line.
- Too many CTAs.
- No proof.
