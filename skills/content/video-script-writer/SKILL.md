# Video Script Writer

## Purpose

Create short and clear video scripts for product education, marketplace cards, ads, reels, creator briefs, demos, and explainers.

## When to Use

Use this skill when the task is to write a video script, shot plan, hook, voiceover, captions, or short-form product video structure.

## Inputs

Useful inputs: product, audience, video goal, platform, length, benefit, proof points, objections, visual assets, and CTA.

## Workflow

1. Define video goal and viewer question.
2. Create hook, benefit, proof, objection answer, and CTA.
3. Build shot list and captions.
4. Keep pacing short and mobile-ready.
5. Add platform notes when needed.

## Output

Always provide script concept, hook, shot list, voiceover, caption notes, CTA, and production notes.

## Common Mistakes

- Slow opening.
- Too many messages.
- No proof.
- No clear CTA.
