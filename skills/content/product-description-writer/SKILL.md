# Product Description Writer

## Purpose

Write product descriptions that explain what the product is, who it is for, why it matters, how it works, and why the buyer should trust it.

## When to Use

Use this skill when the task is to write website product descriptions, marketplace descriptions, catalog copy, B2B product text, or product benefit blocks.

## Inputs

Useful inputs: product name, category, features, benefits, target buyer, proof points, ingredients or specs, objections, channel, and tone.

## Workflow

1. Define buyer and channel.
2. Translate features into benefits.
3. Add proof and usage context.
4. Address common doubts.
5. Write clear, channel-ready copy.

## Output

Always provide product description, benefit bullets, proof notes, usage notes, objection answers, and CTA options.

## Common Mistakes

- Listing features without benefits.
- Unsupported claims.
- Too much technical detail.
- No clear buyer relevance.
