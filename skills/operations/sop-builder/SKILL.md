# SOP Builder

## Purpose

Create simple standard operating procedures for repeatable business tasks.

## When to Use

Use this skill when a task needs clear steps, owner, inputs, outputs, checks, and handoff rules.

## Inputs

Useful inputs: process name, goal, owner, trigger, inputs, tools, steps, quality checks, output, and escalation path.

## Workflow

1. Define process goal and trigger.
2. List required inputs and tools.
3. Break work into clear steps.
4. Add quality checks and owner roles.
5. Define output, handoff, and update rhythm.

## Output

Always provide SOP summary, step list, owner notes, quality checks, handoff rules, and next actions.

## Common Mistakes

- Too many steps.
- No owner.
- No quality check.
- No update rhythm.
