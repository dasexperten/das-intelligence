# Wildberries Card Optimizer

## Purpose

Improve Wildberries product cards for search visibility, shopper clarity, trust, conversion, and category fit.

## When to Use

Use this skill when improving WB titles, characteristics, descriptions, rich content, image order, benefits, proof, reviews, and questions.

## Inputs

Useful inputs: WB SKU, category, current card text, characteristics, images, price, reviews, questions, competitors, search terms, and sales data.

## Workflow

1. Review category, title, characteristics, content, images, reviews, and questions.
2. Identify missing facts, weak benefits, trust gaps, and common doubts.
3. Recommend title, characteristic, description, media, and FAQ changes.
4. Prioritize fixes by search and conversion impact.

## Output

Always provide card diagnosis, title logic, characteristics fixes, media plan, proof notes, doubt fixes, and priority actions.

## Common Mistakes

- Weak characteristics.
- Overloaded title.
- Images that do not explain benefits.
- Ignoring questions and reviews.
