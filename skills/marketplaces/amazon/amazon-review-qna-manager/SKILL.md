# Amazon Review QNA Manager

## Purpose

Use Amazon reviews and Q&A to improve customer answers, listing content, product proof, and trust signals.

## When to Use

Use this skill when analyzing Amazon reviews, preparing answers, reviewing Q&A themes, or improving listing content from customer feedback.

## Inputs

Useful inputs: ASIN, reviews, ratings, Q&A text, product facts, listing content, competitor reviews, and repeated themes.

## Workflow

1. Group reviews and Q&A by theme.
2. Identify praise, complaints, doubts, and content gaps.
3. Draft useful answers using verified product facts.
4. Recommend listing, image, A+ content, or FAQ updates.

## Output

Always provide theme analysis, answer drafts, content gaps, proof notes, update ideas, and next actions.

## Common Mistakes

- Ignoring repeated themes.
- Defensive answers.
- Unsupported claims.
- Not using reviews to improve content.
