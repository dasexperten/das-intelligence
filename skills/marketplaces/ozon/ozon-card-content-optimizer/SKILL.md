# Ozon Card Content Optimizer

## Purpose

Improve Ozon product cards for search, category fit, shopper clarity, trust, media quality, and conversion.

## When to Use

Use this skill when improving Ozon titles, attributes, descriptions, rich content, images, video, reviews, and questions.

## Inputs

Useful inputs: Ozon SKU, category, current card, attributes, media, price, reviews, questions, competitors, search terms, and sales data.

## Workflow

1. Review category, title, attributes, content, images, reviews, and questions.
2. Identify missing facts, weak benefits, trust gaps, and common doubts.
3. Recommend title, attribute, rich content, media, and FAQ changes.
4. Prioritize fixes by search and conversion impact.

## Output

Always provide card diagnosis, title logic, attribute fixes, media plan, proof notes, doubt fixes, and priority actions.

## Common Mistakes

- Weak attributes.
- Generic description.
- Rich content without clear selling logic.
- Ignoring questions and reviews.
