# Data Sync Checker

## Purpose

Check whether data moves correctly between tools, sheets, APIs, marketplaces, CRMs, and reports.

## When to Use

Use this skill when data seems missing, duplicated, delayed, mismatched, or inconsistent across systems.

## Inputs

Useful inputs: source, destination, fields, sample records, sync schedule, filters, logs, expected result, and issue notes.

## Workflow

1. Define source and destination.
2. Compare field map and sample records.
3. Check filters, timing, and duplicates.
4. Identify mismatch cause.
5. Recommend fix and test plan.

## Output

Always provide sync diagnosis, field issues, sample mismatches, likely cause, fix plan, and next actions.

## Common Mistakes

- No sample records.
- No field map.
- Ignoring filters.
- No retest.
