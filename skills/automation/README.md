# Automation Skills

This folder contains practical automation skills for workflow planning, Make scenarios, API integrations, data sync checks, error review, and runbooks.

## Completed Automation Skills

1. `automation-workflow-planner`
2. `make-scenario-planner`
3. `api-integration-planner`
4. `data-sync-checker`
5. `automation-error-reviewer`
6. `automation-runbook-builder`

## Usage Logic

- general automation → `automation-workflow-planner`
- Make.com scenario → `make-scenario-planner`
- API connection → `api-integration-planner`
- data mismatch → `data-sync-checker`
- failed workflow → `automation-error-reviewer`
- operating guide → `automation-runbook-builder`
