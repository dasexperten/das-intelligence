# Customer Journey Designer Checklist

- [ ] Target customer is clear.
- [ ] Desired final action is defined.
- [ ] Journey stages are mapped.
- [ ] Customer questions are listed by stage.
- [ ] Touchpoints are included.
- [ ] Friction points are identified.
- [ ] Proof gaps are named.
- [ ] Post-purchase stage is included.
- [ ] KPIs are tied to each stage.
- [ ] Priority fixes are practical.
