# Conversion Value Tracking Auditor

## Purpose

Audit Google Ads measurement so bidding optimizes toward real business outcomes instead of duplicated, missing, low-quality, or wrongly valued conversions.

## When to Use

Use this skill before Target ROAS, Target CPA, Performance Max scaling, lead-quality optimization, or whenever Ads, GA4, CRM, and revenue numbers disagree.

## Inputs

Useful inputs: conversion actions, primary and secondary goals, values, attribution settings, Google tag or GTM setup, GA4 events, enhanced conversions, CRM imports, transaction IDs, and recent test conversions.

## Workflow

1. List every conversion action and its business purpose.
2. Separate primary bidding goals from secondary observation goals.
3. Check firing, duplication, transaction IDs, value, currency, attribution, and conversion windows.
4. Compare Google Ads, GA4, checkout, and CRM samples.
5. Test new conversions end to end.
6. Recommend a clean bidding goal and value map.

## Output

Always provide tracking diagnosis, conversion action table, value map, duplicate or missing signal notes, test results, and priority fixes.

## Common Mistakes

- Counting page views as primary conversions.
- Importing the same purchase through several sources.
- Giving every lead the same value when quality differs.
- Changing bidding before validation is complete.
