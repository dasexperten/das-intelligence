# Retention Specialist Checklist

- [ ] Retention goal is clear.
- [ ] Target segment is defined.
- [ ] Purchase or usage cycle is known or estimated.
- [ ] Drop-off reasons are identified.
- [ ] Post-purchase touchpoints are mapped.
- [ ] First-use experience is considered.
- [ ] Retention actions match lifecycle stage.
- [ ] Reactivation logic is included.
- [ ] KPIs measure repeat behavior.
- [ ] Next steps are practical.
