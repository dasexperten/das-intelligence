# Retention Specialist

## Purpose

Improve repeat purchase, customer loyalty, reactivation, lifecycle communication, and post-purchase experience.

This skill helps identify why customers do or do not come back, then creates practical retention actions for e-commerce, marketplaces, DTC websites, subscriptions, email, CRM, and B2B relationships.

## When to Use

Use this skill when the task is to:

- increase repeat purchases;
- reduce churn;
- improve post-purchase experience;
- create retention campaigns;
- reactivate inactive customers;
- improve loyalty and customer lifetime value;
- build email, SMS, CRM, or marketplace follow-up logic.

## When NOT to Use

Do not use when the user only needs first-purchase acquisition, a launch plan, or one advertising campaign. Use Growth Strategy Expert if the main bottleneck is not yet known.

## Inputs

Useful inputs:

- product category;
- purchase cycle;
- customer segments;
- repeat purchase data;
- reviews;
- complaints;
- email or CRM data;
- marketplace limitations;
- current post-purchase flow;
- retention goal.

## Workflow

1. Define the retention goal and customer segment.
2. Identify the expected repurchase or usage cycle.
3. Find the main reason customers stop, delay, forget, or switch.
4. Map post-purchase touchpoints.
5. Identify trust, satisfaction, education, and reminder gaps.
6. Create retention actions by lifecycle stage.
7. Add reactivation logic for inactive customers.
8. Define retention KPIs and testing plan.

## Output

Always provide:

- retention diagnosis;
- target segment;
- repeat purchase or usage cycle;
- churn or drop-off reasons;
- lifecycle touchpoints;
- retention actions;
- reactivation ideas;
- KPI table;
- next steps.

## Best Practices

- Retention starts immediately after purchase.
- Teach customers how to get value from the product.
- Use reminders only when timing is relevant.
- Segment by behavior, not only demographics.
- Improve the product experience before adding discounts.
- Measure repeat purchase, not just email opens.

## Common Mistakes

- Treating retention as only email marketing.
- Discounting too often.
- Ignoring first-use experience.
- Sending generic reminders without timing logic.
- Not separating loyal, inactive, and dissatisfied customers.
