# File Folder Organizer

## Purpose

Organize files, messages, screenshots, contracts, invoices, and timelines into a clean folder for business review.

## When to Use

Use this skill when preparing a business file package, partner issue folder, payment history, shipment issue, or document set.

## Inputs

Useful inputs: folder name, date range, documents, screenshots, messages, invoices, payments, timeline, owner, and desired outcome.

## Workflow

1. Define folder name and date range.
2. Group files by document type.
3. Build timeline of events.
4. Identify missing documents.
5. Create folder structure and summary.

## Output

Always provide folder structure, timeline, document list, missing items, owner notes, and next actions.

## Common Mistakes

- Random file names.
- No timeline.
- Missing dates.
- No summary.
