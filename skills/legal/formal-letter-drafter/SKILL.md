# Formal Letter Drafter

## Purpose

Draft clear formal business letters for partners, vendors, buyers, banks, service providers, and official communication.

## When to Use

Use this skill when preparing a formal request, confirmation, notice, reminder, explanation, or response.

## Inputs

Useful inputs: recipient, context, goal, facts, dates, documents, requested action, tone, and deadline.

## Workflow

1. Define recipient and purpose.
2. Summarize facts and dates.
3. State request or response clearly.
4. Add documents or references.
5. Close with next action and deadline.

## Output

Always provide subject, letter body, attachment notes, deadline note, and next action.

## Common Mistakes

- Too emotional.
- Unclear request.
- Missing dates.
- No next action.
