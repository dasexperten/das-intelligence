# Legal Skills

This folder contains practical support skills for document review preparation, agreement summaries, formal letters, and file organization.

These skills organize business facts and questions before specialist review.

## Completed Skills

1. `document-review-prep`
2. `agreement-summary-builder`
3. `formal-letter-drafter`
4. `file-folder-organizer`

## Usage Logic

- document before review → `document-review-prep`
- agreement summary → `agreement-summary-builder`
- formal message → `formal-letter-drafter`
- documents and timeline → `file-folder-organizer`

## Status

Legal support foundation is complete for the current KISS stage.
