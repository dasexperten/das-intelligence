## TONE

- Peer-to-peer, not vendor-to-gatekeeper. We propose value (content, product, data, expertise), not beg for placement.
- First email ≤ 120 words. Editors and webmasters delete long pitches unread.

## CONVERSION GATE — mandatory hard stop

Every partnerships@ email exists for ONE outcome: the recipient says **yes**. Before the confirmation gate, every draft must pass this checklist. Any fail → rewrite, do not send.

1. **Value-first (WIIFM):** the recipient's benefit is stated in the first 2 sentences — before anything about Das Experten. What do THEY get: ready content, unique data, expert commentary, product for review, revenue share? If the first paragraph is about us — fail.
2. **Single low-friction CTA:** exactly ONE ask per email, answerable with one short reply. ❌ «посмотрите наш сайт и подумайте о сотрудничестве» ✅ «пришлю готовый драфт на 800 слов под вашу рубрику X — интересно?». Yes/no question converts; open-ended homework does not.
3. **Effort asymmetry:** we carry the workload (draft ready, images ready, data compiled) — the recipient only approves. State this explicitly in the email.
4. **Proof, not promises:** one concrete credibility marker (existing publication, marketplace rating, clinical data point via product-skill) — never a list of them.
5. **Micro-deadline:** soft urgency in the CTA per Aram's marketing rule («в течение этой недели», «до пятницы», «this week») — no urgency = reply postponed forever = no reply.
6. **Follow-up cadence:** no reply → exactly 2 follow-ups (day 3–4 and day 8–10), each adding NEW value (extra angle, updated data), never «просто напоминаю». After the second silence — thread closed, log and move on. Pushing a third time burns the domain and the persona.

Self-test before sending: **would a busy editor reply to this in under 60 seconds?** If the honest answer is no — the email is not ready.
