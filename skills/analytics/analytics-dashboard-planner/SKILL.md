# Analytics Dashboard Planner

## Purpose

Plan dashboards that show the right numbers for business decisions without visual noise.

## When to Use

Use this skill when creating dashboards for sales, marketing, marketplace, website, operations, logistics, or management review.

## Inputs

Useful inputs: business goal, audience, KPIs, data sources, date range, segments, update rhythm, and decision needs.

## Workflow

1. Define dashboard purpose and audience.
2. Choose useful metrics only.
3. Map data sources and segments.
4. Define layout, filters, and review rhythm.
5. Apply low-noise design and mobile-friendly rules when relevant.

## Output

Always provide dashboard goal, metric list, data sources, layout notes, filters, review rhythm, and next actions.

## Common Mistakes

- Too many charts.
- No decision link.
- No owner.
- No refresh rhythm.
