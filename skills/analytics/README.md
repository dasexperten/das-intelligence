# Analytics Skills

This folder contains practical analytics skills for dashboards, business metrics, ecommerce funnels, marketplace performance, marketing ROI, and report summaries.

## Completed Analytics Skills

1. `analytics-dashboard-planner`
2. `business-metric-analyst`
3. `ecommerce-funnel-analyst`
4. `marketplace-performance-analyst`
5. `marketing-roi-analyst`
6. `report-summary-writer`

## Usage Logic

- dashboard design → `analytics-dashboard-planner`
- general numbers → `business-metric-analyst`
- ecommerce flow → `ecommerce-funnel-analyst`
- marketplaces → `marketplace-performance-analyst`
- marketing spend → `marketing-roi-analyst`
- summary → `report-summary-writer`
