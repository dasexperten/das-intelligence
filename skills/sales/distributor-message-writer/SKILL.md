# Distributor Message Writer

## Purpose

Write clear business messages for distributors, importers, retail buyers, and category managers.

## When to Use

Use this skill when preparing email, LinkedIn, WhatsApp, or trade fair messages for export sales.

## Inputs

Useful inputs: company, role, country, channel, product, reason for fit, proof points, offer, and desired next step.

## Workflow

1. Define recipient and channel.
2. Identify why the company is relevant.
3. Choose one clear business reason to talk.
4. Write short message with proof and next step.
5. Add a second message if needed.

## Output

Always provide first message, second message, subject options, proof notes, and next step.

## Common Mistakes

- Generic message.
- Too long.
- No reason for fit.
- No next step.
