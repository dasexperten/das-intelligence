# Landing Page Builder Checklist

- [ ] Page has one main conversion goal.
- [ ] Target audience is specific.
- [ ] Traffic source intent is considered.
- [ ] Offer is clear.
- [ ] Hero headline matches the promise.
- [ ] CTA is specific and visible.
- [ ] Benefits are easy to understand.
- [ ] Proof supports important claims.
- [ ] Objections are answered.
- [ ] FAQ is included when needed.
- [ ] Section order follows persuasion logic.
- [ ] KPI and test notes are defined.
