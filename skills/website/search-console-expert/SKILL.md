# Search Console Expert

## Purpose

Use Google Search Console data to understand organic search visibility, queries, pages, indexing, CTR, countries, devices, and SEO opportunities.

This skill turns GSC data into practical website, SEO, content, and technical improvement actions.

## When to Use

Use this skill when the task is to:

- analyze Search Console data;
- find pages with low CTR;
- identify keyword opportunities;
- review indexing issues;
- compare country or device performance;
- prioritize SEO fixes;
- connect search queries to page improvements.

## When NOT to Use

Do not use for full technical SEO crawling or schema implementation. Use this skill for GSC-based diagnosis and action planning.

## Inputs

Useful inputs:

- GSC screenshots or exports;
- date range;
- queries;
- pages;
- countries;
- devices;
- impressions;
- clicks;
- CTR;
- average position;
- indexing or coverage issues.

## Workflow

1. Define the SEO question and date range.
2. Review clicks, impressions, CTR, and position trends.
3. Identify high-impression low-CTR queries and pages.
4. Identify ranking opportunities near positions 4–20.
5. Review country, device, and page segments.
6. Check indexing or technical warnings.
7. Recommend title, meta, content, internal linking, and indexing fixes.
8. Prioritize by potential impact.

## Output

Always provide:

- GSC diagnosis;
- query opportunities;
- page opportunities;
- CTR improvement ideas;
- ranking improvement ideas;
- indexing notes;
- internal linking notes;
- priority actions.

## Best Practices

- Focus on impressions plus CTR, not clicks alone.
- Segment by country and device when relevant.
- Improve pages that already have search visibility first.
- Match title and page content to query intent.
- Use internal links to support important pages.

## Common Mistakes

- Ignoring high-impression low-CTR pages.
- Looking only at average position.
- Mixing brand and non-brand queries.
- Not checking countries and languages separately.
- Making SEO conclusions without enough date range context.
