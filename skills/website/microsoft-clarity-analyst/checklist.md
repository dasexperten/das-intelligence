# Microsoft Clarity Analyst Checklist

- [ ] Page goal is clear.
- [ ] Device segment is identified.
- [ ] Heatmap signals are reviewed.
- [ ] Recordings are interpreted as patterns, not one-off proof.
- [ ] Rage clicks are checked.
- [ ] Dead clicks are checked.
- [ ] Scroll depth issues are checked.
- [ ] Mobile behavior is reviewed separately.
- [ ] Fixes are prioritized by frequency and impact.
- [ ] KPI notes are included.
