# Product Page Optimizer Checklist

- [ ] Product page goal is clear.
- [ ] Target buyer is specific.
- [ ] First screen explains product and value.
- [ ] Main benefit is visible.
- [ ] Features are translated into benefits.
- [ ] Proof supports major claims.
- [ ] Price and value are easy to understand.
- [ ] Objections are addressed.
- [ ] FAQ section removes doubts.
- [ ] Product images support selling.
- [ ] CTA is visible and consistent.
- [ ] Priority fixes are practical.
