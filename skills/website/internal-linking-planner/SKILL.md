# Internal Linking Planner

## Purpose

Plan internal links that help users discover relevant pages, support SEO, distribute authority, and connect homepage, category, product, blog, FAQ, and landing pages into a logical website structure.

This skill is built for e-commerce, DTC websites, international websites, content hubs, product catalogs, and SEO-driven sites.

## When to Use

Use this skill when the task is to:

- improve internal linking;
- support SEO priority pages;
- connect content to products;
- improve crawlability;
- reduce orphan pages;
- plan links from blog, FAQ, homepage, category, and product pages;
- improve user navigation between related pages.

## When NOT to Use

Do not use for backlink outreach or technical crawl implementation. Use this skill for internal link planning and page relationship strategy.

## Inputs

Useful inputs:

- website structure;
- priority pages;
- product categories;
- blog or content pages;
- target keywords;
- Search Console data;
- page URLs;
- orphan pages;
- user journey;
- languages or markets.

## Workflow

1. Define priority pages and SEO or conversion goals.
2. Map page types and topic clusters.
3. Identify orphan or underlinked pages.
4. Match informational pages to commercial pages.
5. Recommend anchor text and link placement.
6. Add links to navigation, homepage, product pages, FAQ, and content where relevant.
7. Prioritize links by user value and SEO impact.

## Output

Always provide:

- internal linking diagnosis;
- priority pages;
- link opportunities;
- anchor text recommendations;
- page relationship map;
- orphan page fixes;
- implementation priority list;
- next steps.

## Best Practices

- Link where it helps the user make progress.
- Use descriptive anchor text.
- Connect content pages to relevant product or category pages.
- Support priority pages with links from strong pages.
- Avoid stuffing irrelevant links.
- Plan links by language and market when sites are multilingual.

## Common Mistakes

- Linking only from the menu.
- Using vague anchors like click here.
- Leaving important pages orphaned.
- Linking without relevance.
- Ignoring multilingual page relationships.
