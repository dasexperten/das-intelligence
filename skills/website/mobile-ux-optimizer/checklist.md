# Mobile UX Optimizer Checklist

- [ ] Mobile user task is clear.
- [ ] First screen is understandable.
- [ ] Primary CTA is visible and tappable.
- [ ] Navigation is simple.
- [ ] Text is readable.
- [ ] Tap targets are large enough.
- [ ] Forms are mobile-friendly.
- [ ] Product comparison is easy.
- [ ] Trust signals are visible near decision points.
- [ ] Priority fixes are practical.
- [ ] Mobile KPIs are defined.
