# Homepage Optimizer Checklist

- [ ] Homepage goal is clear.
- [ ] Target visitor is specific.
- [ ] Hero headline explains value quickly.
- [ ] Subheadline adds clarity, not decoration.
- [ ] Primary CTA is obvious.
- [ ] Secondary CTA does not compete.
- [ ] Product categories are understandable.
- [ ] Proof appears near important claims.
- [ ] Trust signals are visible.
- [ ] Main objections are addressed.
- [ ] Section order follows visitor logic.
- [ ] Priority fixes are practical.
