# Checkout Optimizer Checklist

- [ ] Checkout steps are mapped.
- [ ] Total cost is clear early.
- [ ] Delivery time and method are clear.
- [ ] Return policy is visible.
- [ ] Payment security signals are visible.
- [ ] Account creation is not unnecessary friction.
- [ ] Forms are short and clear.
- [ ] Error messages are helpful.
- [ ] Mobile checkout is reviewed.
- [ ] Priority fixes are practical.
- [ ] KPIs measure completed checkout and abandonment.
